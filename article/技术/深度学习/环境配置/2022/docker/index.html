1. 什么是容器

1.1 定义

代码和依赖库的打包集合，让程序可以稳定快速的从一台电脑迁移到另一台。

包含应用运行所需的一切东西：代码、运行时环境、系统工具、库……

1.2 docker image

1.3 WSL2安装

WSL是windows下一个模拟linux的子系统。

win10安装WSL2。

安装后可以在开始菜单搜索WSL启动。

2. Docker hub

https://hub.docker.com/

每一个tag相当于git中一个commit。

在终端输入命令下载image。
